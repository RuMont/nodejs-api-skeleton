RESTful API Endpoints

You may need permission on this server to fetch endpoint data

General

  • get - /api

    Check if server is up

Notes

  • get - /api/notes

    Get all notes. You can specify an id in the url params to get only one entry: /api/notes?id=2

  • post - /api/notes

    Create new notes. You must send an array of notes.

  • post - /api/notes/{id}

    Update notes

  • post - /api/clients

  • post - /api/clients/{id}

    Update clients
